# Text

The `Text` component is one of our most basic primitives, use it and combine it
with other components to build more complex UI patterns.

**Usage:**

1. Import it from `ui`.

```tsx
import { Text } from "ui/text"
```

2. Use it in your components.

```tsx
<Text>This is my Text!</Text>
```

## Variants

While you can leverage and combine the theme tokens to create any text style you
want, we have some pre-built combinations that are used across the UI, to avoid
repeating the pattern on each implementation.

`Text` variants are split in two groups: `ui` (for User Interface texts) and
`doc` (for Document texts).

**Usage:**

```tsx
<Text variant="ui-h1">The Jedi Master trained the young Padawan</Text>
<Text variant="doc-h1">The Jedi Master trained the young Padawan</Text>
```

> The default `Text` variant is `ui-medium`.

## Colors

Just as with the variants; while you can virtually attach any color from the
Theme to a `Text` component via the `css` prop, we took all the colors that
start with `$text-` and added a shorthand prop for calling them. For instance,
if there is a color named `$text-contrast`, the `color` prop you need to pass to
the `Text` component is `contrast`.

**Usage:**

```tsx
<Text color="alt">Alternative Text</Text>
```

> The default `Text` color is `default`
