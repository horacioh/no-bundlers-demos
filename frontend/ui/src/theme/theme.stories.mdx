# Theme

The `Theme` component **must** wrap your application and should be the unique of
its kind in your component tree. The purpose of this component, is to apply some
global styles to your application, and managing light / dark mode theme changes.

**Usage:**

1. Import it from `ui/theme`.

```tsx
import { Theme } from "ui/theme"
```

2. Wrap your application with it.

```tsx
<Theme>
  <MyApp />
</Theme>
```
